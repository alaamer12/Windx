## Summary

This assembly report contains specifications for 16 window units for the Villa Mountain View Hyde Park project in the Fifth Settlement. The windows are manufactured by El Nile Window / Wintech for client Major General Alaa.

**Window Types:**
- Items 1-15: Hinged windows (various configurations with fixed or pleated mesh)
- Item 16: Sliding window (2-sash with mesh)

**Common Specifications:**
- All windows are white colored
- Glass type: Single clear 6mm throughout
- Profile systems: Combined 700 (hinged) and Combined 800 (sliding)
- Various frame profiles including 705, 702, 618, 804, 821, and others
- Iron reinforcements for structural support

**Key Features:**
- Rain milling on top horizontal frames
- Handle milling on operational sashes
- Double beading system for glass retention
- Aluminum tracks for sliding windows
- Fixed and pleated mesh options
